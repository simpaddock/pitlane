About SimPaddock

{{ name }} uses SimPaddock, a MPL-2.0 licensed Open Source league manager.

SimPaddock allows you to manage your league without the need to set up a CMS to get a website for the league.

Key features

  • Automatic calculation of league standings
  • Integrated news feature
  • Driver and team sign up

You can find SimPaddock on Github: https://github.com/simpaddock

Contents

As SimPaddock does not contain any assets, any assets (logos, texts, names) are user published.

Privacy

SimPaddock uses two Cookies for technical purposes:

  • csrftoken - A cookie to help preventing Cross-Site-Request-Forgery
  • sessionid - A cookie to help us to cache requests and improve performance
SimPaddock has no embedded tracking features.
